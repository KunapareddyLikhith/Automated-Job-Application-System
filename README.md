# Automated-Job-Application-System
An AI-powered job application automation system that:

- Analyzes job descriptions
- Calculates profile-job match scores
- Tailors resumes automatically
- Tracks applications
- Integrates with WhatsApp and Telegram job channels
- Uses local LLMs through Ollama

## Tech Stack

- n8n
- Docker
- Ollama
- Qwen
- Python
- PostgreSQL (planned)
