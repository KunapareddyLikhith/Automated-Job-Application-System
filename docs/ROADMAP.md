# Automated Job App System

## Goal

Build an AI-powered job application automation system that:

- Collects job opportunities from WhatsApp and Telegram
- Analyzes job descriptions using local LLMs
- Calculates candidate-job match scores
- Tailors resumes automatically
- Generates ATS-friendly PDFs
- Tracks applications
- Supports manual HR-contact workflows
